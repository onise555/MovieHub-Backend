Movie & Series Streaming Platform  Backend

A clean and scalable backend API for a movie and TV series streaming platform built with ASP.NET Core.
The system provides secure authentication, media catalog management, user interaction features, and video playback services.

Features

JWT-based authentication and authorization

Movie and TV series catalog

Movie and series details, trailers, and metadata

Genres, languages and countries support

User library and wishlist

Reviews and ratings

Watch history tracking

Video player service

Entity Framework Core integration

Modular and extensible architecture

Technologies

ASP.NET Core Web API

Entity Framework Core

SQL Server

AutoMapper

Microsoft Identity

JWT Authentication
